\chapter{Widget catalogue: containers}
\label{ch:catalogue_containers}

\section{card}

A bordered panel with optional header and footer.

\section{panel}

A bordered region with a title. Similar to card.

\section{tabs}

See Chapter~\ref{ch:tabs}.

\section{tab\_strip}

The strip-only variant; content goes elsewhere.

\section{accordion}

Vertical list of collapsible sections.

\section{collapsible}

A single collapsible region.

\section{split}

See Chapter~\ref{ch:splits}.

\section{scroll\_view}

A scrollable container for oversized content.

\section{virtual\_list}

See Chapter~\ref{ch:lists}.

\section{tree\_view}

See Chapter~\ref{ch:trees}.

\section{table}

See Chapter~\ref{ch:tables}.

\section{Recap}

\begin{recap}
\begin{itemize}[leftmargin=*]
  \item Card, panel: bordered visuals.
  \item Tabs, accordion, collapsible: switchable views.
  \item Scroll, virtual\_list, tree, table: scalable
    structures.
\end{itemize}
\end{recap}

\section*{Workshop}

\begin{enumerate}[leftmargin=*]
  \item Build a settings screen using accordion.
  \item Compare card and panel; pick when each fits.
  \item Build a large data view using virtual\_list +
    table.
\end{enumerate}
