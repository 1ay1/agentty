\chapter{Widget catalogue: display}
\label{ch:catalogue_display}

\section{text}

Plain text. Style modifiers attach style.

\section{paragraph}

Multi-paragraph text with wrap.

\section{markdown}

Render Markdown. See Chapter~\ref{ch:markdown}.

\section{code}

Monospace text block, syntax-highlighted if configured.

\section{image}

Inline image. See Chapter~\ref{ch:images}.

\section{block\_image}

Half-block fallback for terminals without image
protocol.

\section{badge}

A small coloured pill with text inside.

\section{kbd}

A keyboard-key display: small box with the key name.

\section{tag}

A coloured label.

\section{avatar}

A square or circular block with initials or image.

\section{separator}

A horizontal or vertical line.

\section{rule}

Like separator but with optional label centred.

\section{Recap}

\begin{recap}
\begin{itemize}[leftmargin=*]
  \item Text, paragraph, markdown, code for prose.
  \item Image, block\_image for graphics.
  \item Badge, kbd, tag, avatar for chrome.
  \item Separator, rule for division.
\end{itemize}
\end{recap}

\section*{Workshop}

\begin{enumerate}[leftmargin=*]
  \item Build an about screen using each display widget.
  \item Mix markdown and code blocks.
  \item Add badges showing version, status, license.
\end{enumerate}
