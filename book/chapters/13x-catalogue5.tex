\section{notification}

A persistent message in a notification area.

\section{alert}

A coloured message bar (info, warning, error).

\section{tooltip}

See Chapter~\ref{ch:zorder}.

\section{popover}

Like tooltip but interactive (can hold buttons, inputs).

\section{snackbar}

A persistent toast at the bottom edge.

\section{help\_text}

A small dim line beneath an input explaining usage.

\section{Recap}

\begin{recap}
\begin{itemize}[leftmargin=*]
  \item Spinners, progress: activity.
  \item Toast, snackbar, notification: messages.
  \item Alert, help\_text: inline information.
  \item Tooltip, popover: hover-revealed.
\end{itemize}
\end{recap}

\section*{Workshop}

\begin{enumerate}[leftmargin=*]
  \item Add feedback for every long-running action.
  \item Use alerts for warnings.
  \item Add help\_text under inputs.
\end{enumerate}
